# Nutrio v11

A comprehensive nutrition tracking and meal planning application.

## Features

- **Loading Screen**: Beautiful animated loading experience
- **Onboarding Flow**: Personalized setup for new users
  - Dietary Style preferences
  - Goal Setting (lose/maintain/gain weight)
  - Body Metrics tracking
  - Dietary Preferences (vegan, vegetarian, etc.)
  - Activity Level assessment
  - Time & Preferences configuration
- **Dashboard**: Rich statistics and visualizations
  - Detailed nutrition charts and graphs
  - Daily recommended meals (365 unique meals)
  - Quote of the day
  - Comprehensive food and calorie tracking
- **Pages**:
  - Achievements (40 unique achievements)
  - My Favourites (recipe management)
  - Goals (detailed progress tracking)
  - Meal Planner (with photo upload support)
  - Meal Analyzer (AI-powered meal analysis)
  - History (complete nutrition history)
  - Account (user profile and settings)

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
